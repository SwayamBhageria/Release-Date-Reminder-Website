About the Extension

Release Date Reminder is a handy Chrome extension designed to help you quickly extract release dates mentioned on web pages, especially from video descriptions or articles. Once a date is found, you can easily create a Google Calendar event to remind yourself.

Key Features

  • Extracts release dates from YouTube video descriptions and other web pages.
  • Context menu option to extract dates from selected text.
  • Automatically attempts to expand YouTube video descriptions for better extraction.
  • Allows manual adjustment of the extracted date.
  • One-click creation of Google Calendar events for reminders.
  • Simple and intuitive interface.
